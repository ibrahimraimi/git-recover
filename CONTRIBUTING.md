# Contributing to git-recover

Thank you for your interest in contributing to `git-recover`!

## Getting Started

1.  Fork the repository.
2.  Clone your fork: `git clone https://github.com/ibrahimraimi/git-recover.git`
3.  Create a new branch: `git checkout -b my-feature-branch`

## Development

### Prerequisites

- Go 1.24 or later

### Running Locally

```bash
go run main.go
```

### Running Tests

```bash
go test ./...
```

### Linting

We use `golangci-lint` for linting.

```bash
golangci-lint run
```

## Pull Requests

1.  Ensure all tests pass.
2.  Ensure the code is linted.
3.  Submit a Pull Request with a clear description of your changes.
